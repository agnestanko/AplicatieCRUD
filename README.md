# Aplicatie CRUD Angular + .NET API

Aplicatie web CRUD pentru gestionarea angajatilor, realizata cu Angular pentru frontend si ASP.NET Core Web API pentru backend. Datele sunt salvate intr-o baza de date SQL Server prin Entity Framework Core.

## Tehnologii folosite

- Angular
- Bootstrap
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- Git si GitHub

## Functionalitati

- Afisarea listei de angajati
- Adaugarea unui angajat nou
- Editarea datelor unui angajat
- Stergerea unui angajat
- Comunicare intre frontend si backend prin API REST
